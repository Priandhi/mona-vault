# 🎮 CTF Commands — Quick Paste Format

Single-line format biar gak ke-truncate pas copy dari markdown.

## 🟢 EASY

```
solve CTF: decode SGVsbG8gV29ybGQ= kasih MD5
```

```
solve CTF: Caesar shift berapa, decode Khoor Zruog
```

```
solve CTF: decode hex 48656c6c6f
```

## 🟡 MEDIUM

```
solve CTF: decode NDg2NTczNmY2NTcyNmQ3Mzc0Nzc2NTczNzQ2NTcy dan identifikasi format
```

```
solve CTF: XOR cipher 1b0e1c1b1a0b0a1c dengan key 0x0f
```

```
solve CTF: Vigenere cipher Wzcpd wgc jg ukqjl dengan key KEY
```

## 🔴 HARD

```
solve CTF: identifikasi format dan decode TVqQAAMAAAAEAAAAA
```

```
solve CTF: stego first letter dari "Hello world from sunny beach today. H-e-l-l-o is the start. e-l-l-o as the middle. m-y-f-r-i-e-n-d-s form the end."
```

## 🎲 RACE

```
solve CTF race: MD5 dari race2024
```

## 💡 Tips Paste

- Single line, no markdown
- Encoded string di-include langsung
- Hindari quotes bertingkat
- Test 1 dulu sebelum kirim banyak
