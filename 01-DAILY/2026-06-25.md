## PWN / Binary Exploitation

### Yang Dipelajari
- ret2libc dengan `leave;ret` dan tanpa `leave` (-O2)
- Self-referencing fake RBP: set RBP ke `buffer_addr + 0x40`
- ASLR: leak + exploit di SATU process yang sama
- `send()` vs `sendline()`: hindari 0x0a di ROP chain

### Key Pitfalls
- `leave;ret` = `mov rsp, rbp; pop rbp` — RBP harus valid
- `ROP(libc).find_gadget()` returns ALREADY REBASED addresses
- Stack alignment: butuh `ret` gadget sebelum `system()` untuk movaps

## RE / Reverse Engineering

### Tools Mastered
- `strings` → hunting flag/decoy strings
- `xxd`/`hexdump` → cari data XOR'd di .rodata
- `objdump -d` → cari `xor $imm`, `cmp`, `call`
- `upx -d` → detect & unpack packed binaries
- `r2 -A -c "afl; pdf"` → basic analysis

### Pipeline
```
file → strings → hexdump → objdump/r2 → decode XOR → verify
```

### Anti-Debug (3 tipe)
1. ptrace (PTRACE_TRACEME)
2. Timing check (execution slowdown)
3. SIGTRAP handler

Bypass: **static analysis** — baca binary tanpa execute
